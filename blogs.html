HOW TO UPLOAD YOUR PROJECT INTO GITHUB USING GIT BASH IN HINDI?

January 3, 2021 by DUO CODERZ

WE AS A TEAM ARE WORKING FOR FOCUSSING ON SUCH TOPICS WHICH ARE HIGHLY UNTOUCHED ON YOUTUBE.

STEPS INVOLVED!!

1.Create a new repository on GitHub. You can also add a gitignore file, a readme, and a license if you want
2.Open Git Bash & Change the current working directory to your local project.
3.Initialize the local directory as a Git repository.
git init
4.Now change the remote origin of you project as follows.
git remote add origin 'your repository url'
5.Check the status of your repositiory as:
git status
6.Now
git pull origin master
7.Now add all the files of your project to the origin as follows.
git add -A
8.Now commit all the added files to your repository
git commit -a -m "first commit"
9.For making new branch do as follows:
git branch branch-name
10.Now for changing the current repository
git checkout branch-name
11.Now final step
git push origin branch-name
